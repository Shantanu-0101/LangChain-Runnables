### LangChain Runnables are core, composable components implementing standard methods (invoke, batch, stream) for building AI workflows via LangChain Expression Language (LCEL).
### Key types:
* RunnableLambda (wraps functions)
* RunnableParallel (concurrent execution)
* RunnablePassthrough (passes inputs through)
* RunnableSequence (chains)
* RunnableBranch (conditional logic)
